糖糖和糖豆｜透镜虚拟实验室

品牌标识
--------
- 页面页头使用净化后的“糖糖和糖豆”品牌图，已移除原图右下角的生成水印。
- 品牌图片位于 assets/tangtang-tangdou-logo-clean.png，网页离线打开时仍可显示。

运行方式：
1. 直接用现代浏览器打开 index.html。
2. 无需联网、安装依赖或执行构建命令。
3. 手机竖屏可继续操作，并会提示横屏获得更清楚的光路视野。
4. 手机打开知识讲解后，上半屏保留实验，下半屏显示同步讲解。

模型约定：
- 使用空气中的近轴厚透镜近似。
- 由折射率 n、两表面曲率半径 R1/R2 和中心厚度 d，通过透镜制造者公式计算等效焦距。
- 演示采用对称双球面透镜，因此 R2 = -R1。
- 再使用 1/f = 1/u + 1/v 和 m = -v/u 计算成像。
- v > 0 表示实像，v < 0 表示虚像。

可调参数与操作：
- 凸透镜或凹透镜。
- 曲率半径 R。
- 折射率 n。
- 中心厚度 d。
- 透镜口径 D。
- 物体高度 h。
- 物距 u 和光屏位置 s。
- 光线模式、焦点、虚光线延长线和距离标注。
- 物体、透镜和光屏均可拖动。

学习功能：
- 知识面板固定在右侧，可关闭；打开时实验和控制区仍可操作。
- 参数变化后，知识面板中的数值、讲解、推演、规律和题目实时同步。
- 每个专用名词按“标准定义、通俗理解与实例、容易混淆”三层讲解。
- 术语使用折叠卡片，默认全部收起，并保留学生手动展开或关闭的选择。
- 不把“正立、倒立、放大、缩小”等容易直接理解的普通描述单独做成术语卡片。
- 页头可统一选择小、中、大字号，演示区、参数区和知识区同步变化，并记住选择。
- 物体、透镜与光屏使用可识别图形，物理标记仅表达焦点、光线和距离。
